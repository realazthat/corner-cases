cubelib_direction_t cubelib_get_direction_by_int3(int x, int y, int z)¶

Returns a direction represented by a given 3 component vector.

Examples:

  • cubelib_get_direction_by_int3(0,+1,0) Returns a direction pointing in the direction of the positive Y axis from the the center of the cube
  • cubelib_get_direction_by_int3(0,0,-1) Returns a direction pointing in the direction of the negative Z axis from the the center of the cube
Note
two of the components must be zero, and one component must be non-zero.
