cubexx::face_t¶

struct cubexx::face_t¶

Represents an face of a cube.

There are 6 faces in a cube. A face_t is 1:1 equivalent to a direction_t; they can be converted from one to the other via face_t::direction() and direction_t::face().

std::uint_fast8_t index() const¶

Retrieve a numeric 0-based index for the face, for use in indexing when storing in an array, or for comparison when storing in a container requiring comparison.

The index will be less than SIZE(). The “null” face has no index, and it is illegal to call index() on it.
