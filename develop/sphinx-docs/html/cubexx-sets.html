struct cubexx::set_base_t¶

CRTP (curiously recurring template pattern) class for a bit-wise set container, where each element is represented by a 0-based index, and the set internally uses a bit representation for the set implementation.

This is just a base class that has most of the implementation. You probably do not want to ever instantiate this directly.

See
corner_set_t, direction_set_t, edge_set_t, face_set_t.
